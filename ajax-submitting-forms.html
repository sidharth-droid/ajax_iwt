What is AJAX Form Submission?

AJAX allows you to submit form data asynchronously without refreshing the page. This results in a smoother user experience by sending the data in the background and receiving the server's response without interrupting the page flow.

Steps for Submitting Forms Using AJAX:

To submit a form using AJAX, follow these steps:

  1. Create an XMLHttpRequest object using new XMLHttpRequest().
  2. Select the form element using document.querySelector() and collect all key-value pairs.
  3. Create a FormData object by passing the form element into the FormData() constructor.
  4. Create a callback function to handle the server's response. This function should be defined inside the onreadystatechange property of the XMLHttpRequest object.
  5. Open the connection using the open() method and specify the POST method and the server URL.
  6. Send the data using the send() method, passing the FormData object.

Example:

This example demonstrates how to collect data from a form, send it asynchronously using AJAX, and display the server's response.


<!DOCTYPE html>
<html>
<body>
<script>
   function sendFormData() {
      // Creating XMLHttpRequest object
      var zhttp = new XMLHttpRequest();
      const mFormEle = document.querySelector("#mForm");
      // Creating FormData object
      const myForm = new FormData(mFormEle);
      // Creating callback function to handle the response
      zhttp.onreadystatechange = function() {
         if (this.readyState == 4 && this.status == 201) {
            document.getElementById("example").innerHTML = this.responseText;
            console.log("Form Data Posted Successfully");
         }
      };
      // Post/Add form data to the server
      zhttp.open("POST", "https://jsonplaceholder.typicode.com/todos", true);
      // Sending the request to the server
      zhttp.send(myForm);
   }
</script>

<!-- Creating simple form -->
<form id="mForm">
   <h2>Enter the requested Data</h2>
   <label for="Utitle">Title</label>
   <input id="Utitle" type="text" name="title"><br>

   <label for="UId">UserId</label>
   <input id="UId" type="number" name="UserID"><br>

   <label for="Ubody">Body</label>
   <input id="Ubody" type="text" name="body"><br>

   <label for="Uage">Age</label>
   <input id="Uage" type="number" name="age"><br>

   <button type="button" onclick="sendFormData()">Submit Record</button>
</form>

<div id="example"></div>
</body>
</html>

Output:

Clicking the "Submit Record" button will trigger the sendFormData() function. The function will send the form data to the server, and upon receiving a successful response, the result will be displayed below the form.
