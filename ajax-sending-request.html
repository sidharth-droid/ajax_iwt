Sending Requests in AJAX

Learn how to use AJAX to send data requests to a server using XMLHttpRequest.

Types of Requests in AJAX

AJAX allows several types of HTTP requests:

  • GET: Retrieve data from the server.
  • POST: Send data to the server.
  • PUT: Update existing data on the server.
  • DELETE: Remove data from the server.

XMLHttpRequest Methods for Sending Requests

To create and send a request, the XMLHttpRequest object uses two main methods:

  • open(method, url, async): Establishes a connection to the server.
  • send(): Sends the request to the server.

The open() Method

The open() method initializes the request by setting the HTTP method (GET, POST, etc.), the URL of the server, and the connection type (synchronous or asynchronous).

Syntax: open(method, url, async)

Parameters:

  • method: HTTP method to be used, e.g., GET or POST.
  • url: URL of the file or API endpoint on the server.
  • async: Set to true for asynchronous, false for synchronous (default is true).

The send() Method

The send() method sends the request to the server. For a GET request, send() is called without parameters. For a POST request, data can be passed as an argument to send(data).

Example: Sending a GET Request

Below is an example of using the XMLHttpRequest object to fetch data from a server.

function showData() {
   let xhr = new XMLHttpRequest();
   xhr.onreadystatechange = function() {
      if (xhr.readyState === 4 && xhr.status === 200) {
         document.getElementById("output").innerHTML = xhr.responseText;
      }
   };
   xhr.open("GET", "https://jsonplaceholder.typicode.com/todos/3", true);
   xhr.send();
}
