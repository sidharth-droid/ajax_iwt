Sending Data Object using AJAX

In AJAX, we can send data objects as part of an HTTP request from a client to a web server. A data object is simply a JavaScript object that contains data in key-value pairs. These objects can represent any type of data, including form inputs, user data, or other information that needs to be sent to the server for processing.

To send a data object using AJAX, we must convert the object into a JSON string using JSON.stringify(). This is necessary because most web frameworks easily support JSON format without additional effort.

Steps to Send Data Object:

  1. Create a new XMLHttpRequest instance: var xhr = new XMLHttpRequest();
  2. Set up the request method and URL using open().
  3. Set the request header using setRequestHeader() to indicate that the request contains JSON data.
  4. Create a data object containing the data to send to the server.
  5. Convert the data object to a JSON string using JSON.stringify().
  6. Send the JSON string using the send() method.

Example Code:

    
       <script>
          function sendDataObject() {
             // Creating XMLHttpRequest object
             var qhttp = new XMLHttpRequest();
          
             // Creating data object
             var myDataObject = {
                "name": "Monika",
                "City": "Delhi",
                "Age": 32,
                "Contact Number": 33333333
             }
          
             // Creating callback function to handle the response
             qhttp.onreadystatechange = function() {
                if (this.readyState == 4 && this.status == 201) {
                   document.getElementById("sample").innerHTML = this.responseText;
                   console.log("Data object sent successfully");
                }
             };
          
             // Open the POST request with the given URL
             qhttp.open("POST", "https://jsonplaceholder.typicode.com/todos", true);
          
             // Set the HTTP request header to specify JSON content
             qhttp.setRequestHeader('Content-type', 'application/json');
          
             // Convert the data object to a JSON string
             var myData = JSON.stringify(myDataObject);
          
             // Send the JSON data to the server
             qhttp.send(myData);
          }
       </script>

How it Works:

When you click on the "Submit" button, the sendDataObject() function is triggered. This function:

  1. Creates a new XMLHttpRequest object.
  2. Defines a data object containing the information to be sent to the server.
  3. Sets up the request with the POST method and the target URL: https://jsonplaceholder.typicode.com/todos.
  4. Sets the content type header to application/json, indicating the data is in JSON format.
  5. Converts the data object to a JSON string using JSON.stringify().
  6. Calls the send() method to send the data object to the server.

Data Object Example:

{
   "name": "Monika",
   "City": "Delhi",
   "Age": 32,
   "Contact Number": 33333333
}

Response from Server:

If the server successfully processes the request, it returns a response, which is displayed in the HTML page. A confirmation message is also logged in the console.

Conclusion:

In this tutorial, we've demonstrated how to send data objects to the server using AJAX. By converting JavaScript objects into JSON strings, we can easily transmit structured data, allowing us to update and process data without refreshing the entire page.
