What is Handling Responses in AJAX?

Handling responses is the process of dealing with the data returned by the server after an AJAX request. Once an AJAX request is sent, the client-side JavaScript waits for the server's response. After receiving the response, it processes the data, updates the page content, or handles errors if any occur.

Steps to Handle Responses:

  1. Create an XMLHttpRequest object using XMLHttpRequest() constructor.
  2. Define an event handler for the onreadystatechange event, which checks the readyState and status of the request.
  3. Open the request using the appropriate HTTP method and URL.
  4. Set any necessary headers (e.g., authentication token).
  5. Send the request to the server using send().

Example:

This example demonstrates how to handle a GET request's response by checking the readyState and status, then displaying the response or an error message.


<!DOCTYPE html>
<html>
<body>
<script>
   function handleResponse() {
       // Creating XMLHttpRequest object
       var qhttp = new XMLHttpRequest();
       // Creating callback function
       qhttp.onreadystatechange = function() {
          if (qhttp.readyState == 4) {
             if (qhttp.status == 200) {
                // Display the response
                console.log(qhttp.responseText)
             } else {
                console.log("Found Error: ", qhttp.status)
             }
          }
       };
       // Open the request
       qhttp.open("GET", "https://jsonplaceholder.typicode.com/todos", true);
       // Send the request
       qhttp.send()
   }
</script>
<h2>Display Data</h2>
<button type="button" onclick="handleResponse()">Submit</button>
<div id="sample"></div>
</body>
</html>

Output:

The response from the server will be logged to the console if the request is successful. If there is an error (e.g., network issue), an error message will appear.

Conclusion

Handling responses in AJAX allows web pages to dynamically update without needing a full page reload. By checking the status of the response, handling errors, and updating the page accordingly, AJAX can create a smooth, interactive user experience.
