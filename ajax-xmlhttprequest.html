AJAX XMLHttpRequest

In AJAX, the XMLHttpRequest object is essential for sending and receiving data asynchronously between the client and server. This allows web pages to update parts of themselves without reloading the entire page, improving the user experience.

XMLHttpRequest can send data in various formats such as XML, JSON, and plain text. It is commonly used in combination with JavaScript to make asynchronous HTTP requests.

XMLHttpRequest Methods

  • new XMLHttpRequest() – Creates a new XMLHttpRequest object.
  • abort() – Cancels the current request.
  • getAllResponseHeaders() – Retrieves all header information of the response.
  • getResponseHeader() – Retrieves a specific header's value from the response.
  • open(method, url, async, user, psw) – Initializes the request with the HTTP method (e.g., GET, POST), the URL, and other optional parameters.
  • send() – Sends the request to the server.
  • setRequestHeader() – Sets custom headers for the request.

XMLHttpRequest Properties

  • onreadystatechange – A callback function that handles changes in the request state.
  • readyState – Holds the status of the request, with values from 0 (uninitialized) to 4 (completed).
  • responseText – Contains the server's response as a string.
  • responseXML – Contains the server's response as XML.
  • status – Contains the HTTP status code of the response (e.g., 200 for success, 404 for not found).
  • statusText – Contains the status message corresponding to the HTTP status code (e.g., OK, Not Found).

Usage of XMLHttpRequest

To use XMLHttpRequest, follow these steps:

  1. Create an XMLHttpRequest object: var xhttp = new XMLHttpRequest();
  2. Define a callback function that will be triggered when the request completes.
  3. Use open() to initialize the request and send() to send it to the server.
