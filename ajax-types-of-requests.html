Types of AJAX Requests

AJAX allows sending different types of requests to interact with web servers dynamically.

Overview

AJAX supports various HTTP request methods to communicate with a server and update web content without reloading the page. The primary request types are:

  • GET: Retrieve data from the server.
  • POST: Send data to the server.
  • PUT: Update data on the server.
  • DELETE: Remove data from the server.

GET Request

The GET request retrieves data from the server, with data typically passed in the URL.

Syntax: open("GET", url, true)

Example: Fetching a record with id = 6 from the server:

function displayRecords() {
   var zhttp = new XMLHttpRequest();
   zhttp.onreadystatechange = function() {
      if (this.readyState == 4 && this.status == 200) {
         document.getElementById("example").innerHTML = this.responseText;
      }
   };
   zhttp.open("GET", "https://jsonplaceholder.typicode.com/todos/6", true);
   zhttp.send();
}

Data will display here after request.

POST Request

The POST request sends data to the server in the request body, often used for creating or submitting data.

Syntax: open("POST", url, true)

Example: Sending data to create a new record:

function sendDoc() {
   var qhttp = new XMLHttpRequest();
   qhttp.onreadystatechange = function() {
      if (this.readyState == 4 && this.status == 201) {
         document.getElementById("postResponse").innerHTML = "Data sent successfully!";
      }
   };
   qhttp.open("POST", "https://jsonplaceholder.typicode.com/todos", true);
   qhttp.setRequestHeader("Content-type", "application/json");
   qhttp.send(JSON.stringify({
      "title": "Sample Title",
      "userId": 1,
      "body": "Sample body content"
   }));
}

PUT Request

The PUT request updates an existing record on the server.

Syntax: open("PUT", url, true)

Example: Updating a record with id = 1:

function updateDoc() {
   var uhttp = new XMLHttpRequest();
   uhttp.onreadystatechange = function() {
      if (this.readyState == 4 && this.status == 200) {
         document.getElementById("putResponse").innerHTML = "Record updated successfully!";
      }
   };
   uhttp.open("PUT", "https://jsonplaceholder.typicode.com/todos/1", true);
   uhttp.setRequestHeader("Content-type", "application/json");
   uhttp.send(JSON.stringify({
      "title": "Updated Title",
      "userId": 1,
      "body": "Updated body content"
   }));
}

DELETE Request

The DELETE request removes a specified record from the server.

Syntax: open("DELETE", url, true)

Example: Deleting a record with id = 2:

function delDoc() {
   var dhttp = new XMLHttpRequest();

      }
   };
   dhttp.open("DELETE", "https://jsonplaceholder.typicode.com/todos/2", true);
   dhttp.send();
}
