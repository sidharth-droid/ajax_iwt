Sending JSON Data using AJAX

AJAX (Asynchronous JavaScript and XML) is a combination of web technologies that allows dynamic updates of web pages by sending and retrieving data from the server without reloading the entire page.

JSON (JavaScript Object Notation) is a lightweight data format used for storing and transferring data. It is easy to read and language-independent. AJAX can transport data in JSON format or plain text. In this tutorial, we will learn how to send JSON data using AJAX.

Steps to Send JSON Data:

  1. Create a new XMLHttpRequest instance: var xhr = new XMLHttpRequest();
  2. Open the request with the open() method and the target URL.
  3. Set the request header using setRequestHeader() to specify that the data is in JSON format (i.e., application/json).
  4. Define a callback function to handle the server's response.
  5. Create the JSON data that needs to be sent.
  6. Convert the JSON data to a string using JSON.stringify().
  7. Send the request with the JSON data using the send() method.

Example Code:

    
       <script>
          function sendDoc() {
             // Creating XMLHttpRequest object
             var qhttp = new XMLHttpRequest();
    
             // Creating callback function to handle the response
             qhttp.onreadystatechange = function() {
                if (this.readyState == 4 && this.status == 201) {
                   document.getElementById("sample").innerHTML = this.responseText;
                   console.log("JSON Data Sent Successfully");
                }
             };
    
             // Open the POST request with the given URL
             qhttp.open("POST", "https://jsonplaceholder.typicode.com/todos", true);
    
             // Set the HTTP request header to specify JSON content
             qhttp.setRequestHeader('Content-type', 'application/json');
    
             // Sending the JSON data to the server
             qhttp.send(JSON.stringify({
                "title": "Mickey",
                "userId": 11,
                "id": 21,
                "body": "Mickey lives in London"
             }));
          }
       </script>

How it Works:

When you click on the "Uploading Data" button, the sendDoc() function is executed. This function:

  1. Creates an XMLHttpRequest object.
  2. Sets up the request method to POST and the target URL: https://jsonplaceholder.typicode.com/todos.
  3. Sets the content type header to application/json, indicating that the data being sent is in JSON format.
  4. Creates a JSON object containing the data to be sent to the server.
  5. Converts the JSON object into a string using JSON.stringify() and sends it to the server with the send() method.

JSON Data Example:

{
   "title": "Mickey",
   "userId": 11,
   "id": 21,
   "body": "Mickey lives in London"
}

Response from Server:

If the server successfully processes the request, it will return a response indicating that a new resource has been created. The response is displayed in the HTML page, and a message is logged in the console.

Conclusion:

In this tutorial, we've learned how to send JSON data using XMLHttpRequest in AJAX. JSON is a popular data format due to its simplicity and ease of use. This method of data transmission is lightweight and efficient, making it a great choice for dynamic web applications.
